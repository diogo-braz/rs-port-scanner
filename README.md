rs-port-scanner
======

`rs-port-scanner` is a simple port scanner written in Rust.

## Installation

```console
$ git clone https://github.com/diogo-braz/rs-port-scanner.git
$ cd rs-port-scanner
$ cargo install --path .
```

## Usage

```
Usage: rs-port-scanner [ip]

Example: rs-port-scanner 127.0.0.1
```
